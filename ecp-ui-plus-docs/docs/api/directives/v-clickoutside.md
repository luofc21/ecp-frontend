# v-clickoutside 外部点击

> 页面中点击元素外部触发绑定事件。

VueUse 已提供相关功能：

- Api 调用，请使用 [VueUse - onClickOutside](https://vueuse.org/core/onClickOutside/)；
- 组件调用，请使用 [VueUse - OnClickOutside](https://vueuse.org/core/onClickOutside/#component-usage)；
- 指令调用，请使用 [VueUse - vOnClickOutside](https://vueuse.org/core/onClickOutside/#directive-usage) 。
