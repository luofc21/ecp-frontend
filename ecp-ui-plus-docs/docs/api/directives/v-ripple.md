# v-ripple 点击波纹

> 可以让任何块级元素产生点击波纹。

```vue
<div v-ripple></div>
```

## 基础用法

使用 `v-ripple` 作用在块级元素上，点击产生波纹。

## 彩色波纹

使用元素 color 值作为波纹的颜色。

## Value

`Object` 配置项，结构如下：

| key | 类型 | 说明 |
| --- | --- | --- |
| center | `Boolean` | 波纹是否居中 |
