# GenerateUUID 生成 UUID

> 根据 UUID 规则生成 UUID 字符串

```js
import { generateUUID } from '@ecp/ecp-ui-plus';
const uuid = generateUUID(prefix)
```

- **入参**
  - `String` prefix: UUID 前缀
- **返回**
  - `String` UUID 字符串
