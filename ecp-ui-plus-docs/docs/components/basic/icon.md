# <ecp-icon> 图标

> 默认引入 [`Ecp-iconfont 图标库`](/ecp-ui-plus/docs/design/iconfont.html) 的图标和 `@element-plus/icons-vue` 的图标。

## Attributes

| 参数 | 说明 | 类型 | 可选值 | 默认值 |
| --- | --- | --- | --- | --- |
| icon | -- | string | -- | -- |
| icon-class | 图标组，用于使用自定义图标 | string | -- | 'ecp-iconfont' |
| element-icon | 是否 element 图标 | boolean | -- | false |
| color | svg 图标填充颜色 | string | -- | -- |
