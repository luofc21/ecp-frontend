# Elevations 阴影

> - 共有 6 种阴影，通过投射方向区分;
> - 使用方法：通过 class 类名使用。

## 阴影类名说明

| 类名 | 说明 | 注意事项 |
| --- | --- | --- |
| shadow-base | 基础阴影; box-shadow 为：$--box-shadow-base | -- |
| shadow-top | 阴影在上; box-shadow 为：$--box-shadow-top; 主要用于固定于页面底下的操作栏 | -- |
| shadow-left | 阴影在左; box-shadow 为：$--box-shadow-left; 主要用于侧边栏 | -- |
| shadow-right | 阴影在右; box-shadow 为：$--box-shadow-right; 主要用于右抽屉 | -- |
| shadow-bottom | 阴影在底部; box-shadow 为：$--box-shadow-bottom; 主要用于悬浮状态下的卡片 | -- |
| shadow-bottom-select | 阴影在底部; box-shadow 为：$--box-shadow-bottom-select; 主要用于下拉框 | -- |
