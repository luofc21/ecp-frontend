# 开发环境配置

## 推荐工具

### Visual Studio Code Visual Studio Code

Free. Built on open source. Runs everywhere.

> [下载链接](https://code.visualstudio.com/)

### 微信开发者工具 微信开发者工具

微信公众号网页调试和小程序开发、调试、代码查看、编辑、小程序预览、发布集成工具。

> [下载链接](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)

### NVM NVM Node Version Manager

NVM is a version manager for node.js, designed to be installed per-user, and invoked per-shell. nvm works on any POSIX-compliant shell (sh, dash, ksh, zsh, bash), in particular on these platforms: unix, macOS, and windows WSL.

> [下载链接](https://github.com/coreybutler/nvm-windows/releases)

## 推荐插件

### PCI IDE Boost PCI IDEBoost

PCI 前端开发辅助工具

> - 下载 PCI IDEBoost (本地运行版 V)
>   - 安装在本地，适用于本地工作区。
> - 下载 PCI IDEBoost (远程辅助版 V)
>   - 安装在远程连接端，适用于 WSL、SSH 连接的工作区，需要配合本地运行版使用。
