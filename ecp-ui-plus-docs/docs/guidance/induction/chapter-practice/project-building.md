# 手摸手教你构建前端项目

## 创建项目

### 第一步：下载并安装插件

- 下载并安装VSCode插件 PCI IDEBoost；

### 第二步：创建新项目

- 打开插件，点击创建新项目：

![创建新项目](../../../assets/images/guidance/induction/pci-ide-boost_project-step-01.png)

- 按提示填写项目基本信息，点击 `完 成` 按钮，完成项目创建：

![填写项目基本信息](../../../assets/images/guidance/induction/pci-ide-boost_project-step-02.png)

- 创建成功后，在 VSCode 中打开项目，开始开发。

## 本地开发

> - 必须安装并启用插件： [ESLint](https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint)、[Prettier ESLint (Vue2)](https://marketplace.visualstudio.com/items?itemName=rvest.vs-code-prettier-eslint)、[Vetur (Vue2)](https://marketplace.visualstudio.com/items?itemName=octref.vetur)、[Vue - Official (Volar) (Vue3)](https://marketplace.visualstudio.com/items?itemName=Vue.volar)；
> - 建议安装插件： [GitHub Copilot](https://marketplace.visualstudio.com/items?itemName=GitHub.copilot)。
