# 公共服务

为产品项目的所有系统提供的基础通用能力。

## 系统管理

提供全局统一的基础系统管理功能，包括：

- 目录管理
- 权限管理
  - 用户管理
  - 用户组管理
  - 部门管理
  - 菜单管理
  - 角色管理
- 字典管理
- 登录管理
  - 在线用户管理
  - 系统登录配置
- 日志管理
  - 用户登录日志
  - 用户操作日志

## 资源管理

提供全局统一的设备资源管理功能，包括：

- 场景管理
- 资源接入管理
  - 资源接入管理
  - 资源档案定义
  - 接入平台管理
- 档案信息共享
