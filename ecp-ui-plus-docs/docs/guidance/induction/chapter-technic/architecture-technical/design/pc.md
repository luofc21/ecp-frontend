## 工程基础

### Vue2 版

大部分现存 PC 端项目使用这个模板工程创建，工程仓库：<http://git.pcitech.com/frontend/template-vue>。

基于 Vue2 全家桶搭建的 PC 端模板工程，采用 [Qiankun](https://qiankun.umijs.org/zh) 微前端架构，适配 Node10.15.3。

关键依赖：[Vue 2](https://v2.cn.vuejs.org/)、[Vue-cli](https://cli.vuejs.org/zh/)、[Webpack](https://webpack.docschina.org/)、[Vue-router](https://v3.router.vuejs.org/zh/)、[VueX](https://v3.vuex.vuejs.org/zh/)、[Axios](https://axios-http.com/zh/docs/intro)、[Qiankun](https://qiankun.umijs.org/zh)、[Element-ui](https://element.eleme.cn/#/zh-CN)、[Ecp-ui](http://frontend.pcitech.online/ecp-ui/#/installation)。

具体使用指南及注意事项见 [中后台项目的模板工程](/docs/template-vue/)。

### Vue3 版

目前处于推广阶段，工程结构与 Vue 2 版基本相同。

基于 Vue3 全家桶搭建的 PC 端模板工程，采用 [Wujie](https://qiankun.umijs.org/zh) 微前端架构。

关键依赖：[Vue 3](https://cn.vuejs.org/)、[Vite 3](https://cn.vitejs.dev/)、[Rollup](https://cn.rollupjs.org/)、[Vue-router](https://router.vuejs.org/zh/)、[Pinia](https://pinia.vuejs.org/zh/)、[Axios](https://axios-http.com/zh/docs/intro)、[Wujie](https://wujie-micro.github.io/doc/)、[Element-plus](https://element-plus.org/zh-CN/#/zh-CN)、[Ecp-ui-plus](http://frontend.pcitech.online/ecp-ui-plus/)。
